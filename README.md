# discord_rpg_bot
RPG battle bot for discord

## Commands
### Admin
#### !battle start
#### !battle stop
#### !create <user>
#### !delete <user>

### Out of battle
#### !create
#### !delete
#### !join
#### !stats
#### !shop
#### !buy
#### !items
#### !equip

### In battle
#### !attack
#### !defend
#### !magic
#### !item
